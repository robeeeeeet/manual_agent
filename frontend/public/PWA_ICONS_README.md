# PWA アイコン設定ガイド

PWAアプリのアイコンを設定するために、以下のファイルを用意してください。

## 必要なアイコンファイル

1. **icon-192.png** (192x192 px)
   - ホーム画面に追加時に表示されるアイコン

2. **icon-512.png** (512x512 px)
   - スプラッシュスクリーンなどで使用される高解像度アイコン

3. **apple-touch-icon.png** (180x180 px)
   - iOS Safari でホーム画面に追加時に使用されるアイコン

## アイコンの作成方法

### オプション1: オンラインジェネレーターを使用

1. https://realfavicongenerator.net/ にアクセス
2. 1024x1024 px の PNG 画像をアップロード
3. 必要なサイズのアイコンを一括生成してダウンロード
4. 生成されたファイルを `frontend/public/` に配置

### オプション2: 画像編集ツールを使用

1. Figma、Canva、Photoshop などで以下のサイズの画像を作成:
   - 192x192 px → icon-192.png
   - 512x512 px → icon-512.png
   - 180x180 px → apple-touch-icon.png
2. 作成した画像を `frontend/public/` に配置

## デザインガイドライン

- **背景色**: 白 (#ffffff) または透過推奨
- **メインカラー**: 青 (#3b82f6) - アプリのテーマカラーに合わせる
- **シンプルなデザイン**: 小さいサイズでも認識しやすいアイコン
- **推奨モチーフ**: 説明書、メンテナンス、チェックリストなど

## 配置後の確認

アイコンを配置した後、以下を確認してください:

1. ブラウザで `http://localhost:3000/manifest.json` にアクセスしてmanifestが正しく読み込まれているか確認
2. Chrome DevTools の Application タブで Manifest セクションを確認
3. iOS Safari や Android Chrome で「ホーム画面に追加」を試す
