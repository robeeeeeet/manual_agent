## 開発フェーズ

### Phase 0: フィジビリティ確認 ⚠️ 最重要

このアプリの成否を決めるAI機能の実現可能性を先に検証する。

#### 0-1. 画像からのメーカー・型番読み取り
- [x] 家電の写真（ラベル含む）をGeminiに送信
- [x] メーカー名・型番を正確に抽出できるか検証
- [x] 外観から商品を推測し、ラベル位置を指示できるか検証
- [ ] 複数の家電カテゴリでテスト（エアコン、洗濯機、冷蔵庫など）

#### 0-2. メーカー・型番からマニュアルPDF取得
- [x] メーカー名・型番からWeb検索でPDFを見つけられるか
- [x] Gemini + Web検索の組み合わせで自動取得が可能か
- [ ] 取得成功率の検証（主要メーカー）

#### 0-3. マニュアルからメンテナンス項目抽出
- [x] PDFをGeminiに読み込ませてメンテナンス項目を抽出
- [x] 周期（月1回、年1回など）を正確に抽出できるか
- [ ] 複数のマニュアルでテスト

#### Phase 0の成果物
- 各検証の成功率レポート
- 実装上の課題と対策
- Go/No-Goの判断
